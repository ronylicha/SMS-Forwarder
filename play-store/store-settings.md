## Declaration d'utilisation de SMS/MMS

Google requiert une declaration specifique pour les applications utilisant les permissions SMS.
L'application doit etre declaree comme cas d'utilisation approuve :

**Type d'utilisation** : SMS/MMS transfer/forwarding
**Justification** : L'application est un outil de transfert SMS dont la fonctionnalite principale est de rediriger automatiquement les messages entrants vers un numero choisi par l'utilisateur. C'est le cas d'utilisation declare et exclusif de l'application.
